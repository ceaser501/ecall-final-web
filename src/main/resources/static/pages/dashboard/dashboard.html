    개발중

    상황 심각도 자동 분류 (1-5단계)
    개발예정
  • 2-3. 음성 톤 분석

    개발예정

  • 3-2. 지도 시각화

    개발예정
  • 3-3. 경로 분석
    최적 접근 경로 계산
    개발예정
4 텍스트 처리 및 요약
  • 4-1. 대화 내용 요약
    핵심 내용 자동 요약 생성
    개발중
  • 4-2. 핵심 정보 추출

    개발예정
  • 4-3. 보고서 자동 생성
